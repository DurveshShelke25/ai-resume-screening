# 🚀 AI Resume Screening System

## 📌 Project Description

The **AI Resume Screening System** is a web-based application that analyzes resumes and matches them with job descriptions using intelligent skill-based matching.

This system helps recruiters quickly identify suitable candidates by calculating a match score and highlighting matched and missing skills.

---

## 🎯 Features

* 📄 Upload Resume (PDF format)
* 🧠 Skill-based matching algorithm
* 📊 Match score calculation
* ✅ Matched skills display
* ❌ Missing skills display
* 🎨 Modern user interface

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python (Flask)
* **Libraries:** PyPDF2, Flask-CORS
* **Concepts Used:** NLP, Skill Matching

---

## ⚙️ How It Works

1. User enters job description
2. Uploads resume (PDF)
3. System extracts text from PDF
4. Skills are matched with predefined skill set
5. Match score is calculated
6. Results are displayed with matched & missing skills

---

## ▶️ How to Run the Project

### 1️⃣ Install Dependencies

```
python -m pip install -r requirements.txt
```

### 2️⃣ Run Backend

```
cd backend
python app.py
```

### 3️⃣ Run Frontend

Open `index.html` in browser or use Live Server
